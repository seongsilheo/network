1. 컨테이너 목록 출력

        docker ps
      
<br/><br/>

2. 컨테이너 스탑

       docker stop <컨테이너 이름>
    
<br/><br/>

3. 컨테이너 삭제

       docker rm <컨테이너 이름>
    
<br/><br/>

4. 스크립트 파일 생성

       touch <이름>.sh / vi <이름>.sh
    
<br/><br/>

5. 스크립트 파일 권한 설정 (+x는 모든사용자에게 실행권한 추가)

       chmod +x <이름>.sh
    
<br/><br/>

6. sh파일 수정  (

       vim <이름>.sh
    
<br/><br/>

7. sh파일 실행

       ./<이름>.sh
    
<br/><br/>
